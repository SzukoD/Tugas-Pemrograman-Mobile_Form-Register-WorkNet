# Aplikasi Form Register WorkNet

## Pemrograman Mobile 1

### Deskripsi Project

Project ini merupakan aplikasi **Form Register Android** yang dibuat untuk memenuhi tugas mata kuliah **Pemrograman Mobile 1**.
Aplikasi ini berisi form registrasi lengkap dengan validasi, selection controls, spinner data, dialog konfirmasi, dan gesture interaction (Long Press).

### Fitur yang Dibuat

#### 1. Complete Form

Form registrasi berisi:

* Nama Lengkap
* Email
* Password
* Confirm Password

---

#### 2. Advanced Validation

Validasi yang digunakan:

* Tidak boleh kosong
* Format email harus valid
* Password minimal 6 karakter
* Password dan Confirm Password harus sama
* Validasi real-time (merah jika salah, hijau jika benar)

---

#### 3. Selection Controls

Menggunakan:

* **RadioGroup** → Jenis Kelamin
* **CheckBox** → Hobi (minimal pilih 3)

---

#### 4. Spinner & Dialog

Menggunakan:

* **Spinner** untuk memilih Provinsi dan Kota
* **AlertDialog** untuk konfirmasi saat submit

---

#### 5. Gesture Interaction

Menggunakan:

* **Long Press pada tombol Register** untuk submit form
* Klik biasa hanya menampilkan informasi
* Tekan lama untuk mengirim data

---

#### 6. GitHub Repository

Project ini diupload ke GitHub sesuai dengan instruksi tugas.
